# KR 브리프 — 프로그램 매매 + 기술적 분석/전략 추가 설계

작성일: 2026-07-23 · 대상 파이프라인: `fdo2a/fdo2a.github.io` (KR 저녁 마감브리프)

## 배경·목표

KR 저녁 마감브리프에 두 가지를 추가한다.
1. **프로그램 매매** — 차익/비차익/전체 순매수(코스피·코스닥)를 수급 섹션에 통합.
2. **기술적 분석 & 시나리오 전략** — 4종(코스피·코스닥·삼성전자·SK하이닉스)에 이평/볼린저/일목 기반 레벨 + 조건부 트레이딩 시나리오.

핵심 원칙: **수치는 전부 결정론적으로 계산(데이터 수집기)**, writer는 계산된 레벨 위에서만 서술·시나리오를 짠다. 수치 창작 절대 금지(기존 팩트체크 게이트 연장).

## 결정 사항 (사용자 확정)

- 프로그램: 수급 섹션 **내 서브블록**, **순매수 3종만**(차익·비차익·전체), 코스피+코스닥.
- 기술적: **4종 전부**, 이평 **20·60·120일**(5일 제외), 볼린저 **20일±2σ**, 일목 **9/26/52**(표준), **캔들 차트에 오버레이**.
- 시나리오: 종목당 **상승/중립/하락 3종**, 표 형식(트리거·목표·무효화·근거).
- 적용: 오늘치(2026-07-23) 발행본에 **새 섹션만 삽입**해 재발행(전체 재작성 아님). 이후 루틴부터 상시 반영.

## A. 프로그램 매매

### 수집
- `scripts/kr/sources.py`: `fetch_program_flows(sosok, bizdate)` — `finance.naver.com/sise/programDealTrendDay.naver?bizdate=&sosok=`(01=KOSPI, 02=KOSDAQ). 단위 **억원**(flows와 동일, 페이지 `단위:억원` 확인).
- `scripts/kr/program.py`(신규, 순수 파서): 테이블 컬럼 `[날짜, 차익매수, 차익매도, 차익순매수, 비차익매수, 비차익매도, 비차익순매수, 전체매수, 전체매도, 전체순매수]` → rows(10거래일). `flows.flows_freshness` 재사용해 신선도 라벨.
- 산출 `kr/data/kr_program.json` — `{KOSPI:{rows,latest_date,flows_date,flows_provisional,label,stale}, KOSDAQ:{...}}` (flows.json 동형). writer는 당일(row[0])만 사용.

### 리포트 (§4 수급 서브블록)
- 수급 표/문단 아래, 거래대금 상위 섹션 앞에 삽입.
- 시장별 표: 차익 순매수 / 비차익 순매수 / 전체 프로그램 순매수(억원) + 신선도 태그.
- 해석 1문단: 비차익(방향성 바스켓·기관/외국인 의도) vs 차익(선물 연계 기계적), 외국인 방향과 정합/괴리.

## B. 기술적 분석 & 시나리오 전략

### 계산 (`scripts/kr/technical.py`, 신규 순수 함수)
입력: yfinance OHLC(charts용으로 이미 받는 2년치) DataFrame. 종목별 산출:
- 이평 20/60/120 + 정배열/역배열/혼조 판정, 현재가 대비 이격(%).
- 볼린저 20±2σ: 상·중·하단, %b, 밴드폭.
- 일목: 전환선(9), 기준선(26), 선행스팬1=(전환+기준)/2 → 26 시프트, 선행스팬2=(52고+52저)/2 → 26 시프트, **현재 시점 구름 상·하단**(26봉 전 값이 오늘에 투영됨에 주의), 가격의 구름 위/속/아래 판정, 후행스팬.
- 스윙 고·저점(최근 N봉) — 시나리오 목표/손절 레퍼런스.
- → `kr/data/kr_technical.json` (4종). **모든 값 계산값.**

### 차트 오버레이 (`scripts/kr/charts.py` 확장)
- 기존 4종 캔들에 이평선(20/60/120)·볼린저밴드(상/중/하)·일목 구름대(선행 26봉 시프트) 오버레이. → `kr_charts.png`.

### 리포트 (일봉 차트 바로 아래 신설 섹션)
**최종 형태(2026-07-23 사용자 지시): 표 제거, 산문.** 오버레이 차트(§3 일봉 차트) 바로 아래에 종목별 **산문 1문단** × 4종. 각 문단은 현재 위치(MA20/60/120·구름 대비, 정/역배열)를 짚고 조건부 전략을 서술 — 상승 트리거(저항 회복/돌파)→목표(다음 저항·스윙 고점), 하락 트리거(지지 이탈)→목표(다음 지지·스윙 저점)·최후 지지선. **레벨은 kr_technical.json의 계산값·스윙 고저만 인용, 창작 금지.**

성격: buy-side 조건부 기술 셋업(시장 코멘터리), 개인화 투자권유 아님.

> 초기 설계는 통합 레벨 표 + 종목별 시나리오 표였으나, 사용자 요청으로 차트 밑 산문으로 전환(표 제거).
